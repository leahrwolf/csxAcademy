Variables

Unlike other languages, pythons variables are a little funky. To create a variable, type x = "X", then save. Congrats! You just made a string variable!

Now type print x into your code. Save again, and test it out in the terminal. It should print X.

Variables in python are sort of like variables in math. Like, X = 23 in 2X + 3.

Now lets try changing up that variable. After your current code, type x = len(x) This will change the variable x to the number of characters in the variable x, so in this case, one. After saving, it should print
