How do I start using Html?

You can start by downloading sublime, a program that allows you to write many different programs. If you've seen the python page, you'll reconize this program. So, if you already have it, you don't need to download it again! I won't be covering how to use the basic functions of sublime in this section, see Starting with Python, for that.

external media

Creating the file

Whenever you start a new Html program in sublime, you will first need to make a file, and save it as whatever you want to call it .html.

Exsample: programname.html

The format

Every html program on sublime starts with a specific format.

<!DOCTYPE html>

<html>

<head>

</head>

<body>

</body>

</html>
