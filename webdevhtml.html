How do I start using HTML?

You can start by downloading sublime, a program that allows you to write many different programs. If you've seen the python page, you'll reconize this program. So, if you already have it, you don't need to download it again! I won't be covering how to use the basic functions of sublime in this section, see Starting with Python, for that.



W3schools

Creating the file

Whenever you start a new Html program in sublime, you will first need to make a file, and save it as whatever you want to call it .html.

Exsample: programname.html

The format

Every html program on sublime starts with the same specific format.

<!DOCTYPE html>

<html>

<head>

</head>

<body>

</body>

</html>

No terminal

Unlike python, Html does not open program through the terminal, reather through your web broweser. But don't worry! Just beacause you make an html file, doesn't mean it's on the internet now for everyone to see! It's just a file, and you will need a bit more to get it online. Now that you have the format down, go into your files and select the html file you are working on. It should pop up in your browser. If you want to open a file in sublime, right click (or do an alternate form of right clicking) and select Open with. Sublime should be one of the options.

The head

The first bit we need to look at is the head. One of the things you can do there is change the little tab at the top of your browser's name. Press enter from the first bit of the head, then tab in.

Type <title>My Webpage</title>

It should look like this.

<head>

*tab in css*<title>My Webpage</title>

</head>

You can change "My Webpage" to anything you want, really. There are many more things you can do with the head, but we won't be covering them right now.

The Body

Now that we have the head, all we need is the body. Usally, the body has more code in it.

You can tab and enter your way inbetween <body> and </body>, like you did in the head.

Headings

Let's start off by creating a heading. Type this:

<h4>My Webpage</h4>

h4 is the heading we are using. h3, h2, and other numbers are differnt heading sizes. Feel free to mess around with it, and as always, you can change the heading to whatever you want.

Paragraphs and Normal text

Like what I just did, you can add paragraphs, or spaces in your text by simply adding a <p> at the start, and a </p> at the end. You'll end up with spaces in the text,

Like this!

Ordered and Unordered lists

In html, you can create lists of things. There are two types of ways to do this. Ordered and unordered lists.

Ordered lists list whatever you want in numbers, starting with one and moving down the list. Unordered lists are just bullet points.

To make an unordered list, first type <ul> then press enter, then type </ul>. Go back up to <ul> and press enter.

Now, add <li>Cats</li>

You can press enter again, and type <li>Dogs</li>

You can keep doing this for as long as you want, with whatever you want.

An ordered list is the same thing you just did, but with <ol> and </ol> instead of <ul> and </ul>

Links

You can set up links to websites or other html files on your computer, like here. Just type

<a href-"http://www.wikipedia.org" target="_blank>Look at my link!<a>

The link can be anything you want it to be.
