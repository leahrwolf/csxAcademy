What is Python?

Python is a coding language, like Javascript, CSS, or Html. It is a great language for learning how to code. Once you have learned one language, it makes it much easier to learn another.

external media

How can it be used?

Python can be used to make small, text based adventure games in the terminal of your computer, to making large, complex animations in Blender (A 3D animation program).

Python Vs. Other Coding languages

Python is very different from other popular coding languages. This does not mean it is a huge hassle to learn another once you have learned python, though. Python is much simpler than the larger, more complex programs. It is a good launchpad into the world of coding.
