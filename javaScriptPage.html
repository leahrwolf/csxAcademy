What is Javascript?

Javascript is a HTML programming language that has the computer do what the user tells it to do.

The Basics to Javscript

Not to be confused with Java, another programming langauge, Javascript is a programming language that allows the user to interact with the webpage.

Javascript can be placed anywhere in the body or the head of an HTML page. To do you first type <script>, followed by your code, then </> to close your tag.

A function of Javascript is a piece of the Javascript code. They are executed when the user asks for the function. Javascript can be used for detecing the browser the user used to visit the web page.

A string in Javascript is used to store text usually inserted by the user. A statement in Javascript is used as an instruction for the web browser. A variable in Javascript, similar to a variable in Python, is something that stores value of data. To create a variable you first type var, you then type your variable name, let us call it z, you than make something equal to the variable let us call it 75. This would be written as var z=75.
