    The Terminal

    For this tutorial, I’ll be using the MacBook Pro’s default terminal. It comes with the laptop, and requires no download. You can find it in your programs. There are certain commands for using the terminal. You can use them by typing them in and pressing enter.

    LS: Show everything in the current location. It should list everything. CD ___: Open the folder specified after CD. Ex: “CD Documents” python ____.py: Activates the python program specified in ____.

    Use these commands to navigate your computer until you find the program itself. Activate it, and it should print “Hello, world!” If it pops up with an error, or something else, check your code, or see if you saved.

    Variables

    Unlike other languages, python’s variables are a little funky. To create a variable, type x = “X”, then save. Congrats! You just made a string variable!

    Use these commands to navigate your computer until you find the program itself. Activate it, and it should print Hello, world! If it pops up with an error, or something else, check your code, or see if you saved.

    Now type print x into your code. Save again, and test it out in the terminal. It should print X.

    Variables in python are sort of like variables in math. Like, X = 23 in 2X + 3.

    Now lets try changing up that variable. After your current code, type x = len(x) This will change the variable x to the number of characters in the variable x, so in this case, one. After saving, it should print
